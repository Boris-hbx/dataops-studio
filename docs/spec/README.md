# Spec 文件夹

存放所有经 Agent 团队（`/team-baoxing`）或手动产出的 **需求规格 (Spec)** 文档。

## 命名规范

```
SPEC-{序号}-{简短标题}.md
```

示例: `SPEC-001-data-lineage-page.md`, `SPEC-002-mcp-integration.md`

## Spec 模板

每个 Spec 文件必须包含以下章节：

```markdown
## Spec: {任务标题}

### 功能描述
### 用户故事
### 验收标准
### UI 行为要点
### API 需求
### 待澄清项（如有）
```
